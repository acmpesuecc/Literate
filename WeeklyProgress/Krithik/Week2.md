## Understanding the Project
I started by trying to clearly understand what the Literate project is about.

From what I learned, Literate is a Progressive Web App (PWA) PDF reader. The app aims to provide a smooth and easy reading experience on all devices. Users can view, bookmark, and annotate PDF documents. The project also supports keeping track of reading progress, bookmarks, and annotations. This way, users can continue reading without interruption on any platform. The main focus is on simplicity and usability(This is exactly what I wrote in the README.md file)

Understanding this helped me explain the project better and avoid confusion.

## Learning About the Technologies
I spent time learning about the main technologies used in the project and why they are needed.

- **React** is used to build the user interface which is technically the UI of the front end.
- **Vite** This thing helps with faster developement.
- **MuPDF using WebAssembly** Renders the file faster in a browser.
- **IndexedDB** Allows to store the PDF in the storage so that we can access it even without internet
- **Service Workers** help the app work offline and improve performance.
- **OAuth 2.0 with Google Sign In** is used to let users log in securely using their Google account.
- **Zustand** is used to store the information about the changes made on the pdf is what i understood for now.

Learning about these helped me understand how the app works as a complete system.
